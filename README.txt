REQUIREMENTS :

Visual Studio 2010, Microsoft Windows Server 2008

Other Requirements

.Net Framework 3.5
IIS 7
UDDI SDK 2.0



The web services need to be deployed on windows server 2008

Running the web portal

1. Start the web services (webservice2, Ebay_finding and Ebay_singleItem) within windows server 2008
2. Make sure localhost/uddi is available ( UDDI is installed and running within the server)
3.From the Client, Run Default.aspx from Website8 which is the client interface.
